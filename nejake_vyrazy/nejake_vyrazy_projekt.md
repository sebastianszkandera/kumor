# Nějaké výrazy

## Popis a cíl projektu
Projekt obsahuje ukázky práce s regulárními výrazy v Pythonu. Cílem je mít samostatný adresář pro demonstraci pattern matchingu a práce s řetězci.

## Popis funkcionality programu
- Obsahuje skripty pro testování různých regulárních výrazů
- Demonstruje validaci textu, vyhledávání vzorů a nahrazování
- Slouží jako učební materiál pro práci s modulem `re`

## Technická část
- `re` modul standardní knihovny Pythonu
- Skripty mohou obsahovat příklady spojení, skupin a modifikátorů
- Data se zpracovávají na úrovni řetězců

## Spuštění
```bash
python nejake_vyrazy\regularni_vyrazy.py
python nejake_vyrazy\regularni_vyrazy_dva_tecka_nula.py
```

## Poznámky
Doporučeno doplnit komentáře do jednotlivých skriptů a uvést příklady vstupních textů, které ověřují chování regulárních výrazů.
