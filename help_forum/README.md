# Help Forum

To je proste forum appka v Pythonu. Umoznuje pridavat otazky a odpovedi.

## Co dela

- Pridavat otazky
- Pridavat odpovedi k otazkam
- Zobrazovat otazky a odpovedi
- Uklada data do JSON souboru

## Pozadavky

- Python 3.x
- Tkinter (je tam defaultne)

## Spusteni

python help_forum.py

## Jak pouzivat

1. Spust appku
2. Napis otazku do pole dole a klikni "Pridat otazku"
3. Klikni na otazku vlevo
4. Napis odpoved a klikni "Pridat odpoved"
5. To je vse

## Ulozeni dat

Vsechno se uklada do forum_data.json. Pri dalsim spusteni se nacte.

## Chyby

Pokud neco nejde, ukaze se chyba. Napr. prazdna otazka nebo nevybrana otazka pro odpoved.
