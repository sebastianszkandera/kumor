# Kumor

## Popis a cíl projektu
Projekt představuje testovací repozitář s jednoduchým Python kódem. Cílem je mít centrální místo pro ověřování funkcí a experimenty s projekty v tomto repozitáři.

## Popis funkcionality programu
- Obsahuje malý Python skript `pp.py`
- Slouží jako testovací základna pro vývoj a ladění kódu
- Repozitář obsahuje příklad jednoduchého `README`

## Technická část
- Python 3
- Jednoduchý soubor `pp.py` s testovacím kódem
- Součástí projektu je vlastní dokumentace a popis v `README`

## Spuštění
```bash
python kumor\kumor\pp.py
```

## Poznámky
Tento projekt je primárně určen pro interní testování a práci s repozitářovou strukturou. Neukládají se zde žádné osobní údaje.
