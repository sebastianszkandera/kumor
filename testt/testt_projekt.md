# Testt

## Popis a cíl projektu
Tento projekt slouží jako jednoduché testovací prostředí pro Python skripty. Cílem je mít oddělený adresář pro experimenty a krátké ukázky kódu.

## Popis funkcionality programu
- Obsahuje jediný soubor `testt.py`
- Může být používán pro rychlé ověření nápadů a funkcí
- Slouží jako sandbox pro testování syntaktických a logických změn

## Technická část
- Python 3
- Pokud je to potřeba, lze přidat další závislosti přímo do tohoto adresáře
- Projekt nevyžaduje žádné speciální knihovny v současném stavu

## Spuštění
```bash
python testt\testt.py
```

## Poznámky
Doporučeno používat tento adresář pro rychlé experimenty bez dopadu na hlavní projekty.
