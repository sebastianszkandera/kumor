# Bordel

## Popis a cíl projektu
Tento projekt slouží jako experimentální sbírka menších Python skriptů a her. Cílem je mít jeden samostatný adresář se všemi ukázkami a rychlými prototypy.

## Popis funkcionality programu
Adresář obsahuje několik různých souborů se hrami a testovacími skripty, například jednoduché 2D hry nebo grafické ukázky.

## Technická část
- Python 3
- Různé knihovny závisí na konkrétním souboru, například `pygame` nebo `tkinter`
- Každý soubor je samostatná ukázka
- Projekt slouží jako místo pro experimenty a rychlé ověření nápadů

## Doporučené soubory
- `simple_3d_game.py` - demonstrační herní smyčka
- `snakegame.py` - jednoduchý had
- `story_game.py` - textová hra
- `test.py` - testovací skript

## Spuštění
Spusťte konkrétní soubor podle jeho funkce, například:
```bash
python bordel\snakegame.py
```

## Poznámky
Tento projekt je určený pro interní testování a vývoj. Okremě testovacího kódu je dobré použít smysluplné komentáře v každém skriptu.
